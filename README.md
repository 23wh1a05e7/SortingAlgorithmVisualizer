# Sorting Algorithm Visualizer

A simple interactive sorting algorithm visualizer built with HTML, CSS, and JavaScript.Developed an interactive web-based application to visualize the working of different sorting algorithms in real time. The project helps users understand how sorting techniques operate step-by-step through animated graphical representations.

Implemented popular algorithms such as Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, and Quick Sort with adjustable array sizes and sorting speeds. The visualizer provides a clear comparison of algorithm efficiency, time complexity, and behavior during execution.

Built using technologies like React / HTML, CSS, and JavaScript to create a responsive and user-friendly interface.

## Features

-Real-time visualization of sorting processes
-Speed control for animations
-Random array generation
-Comparison of multiple sorting algorithms
-Interactive and responsive UI
-Display of time and space complexities

## How to Run

1. Open `index.html` in your browser.
2. Watch the sorting animation.

## Purpose

The project is designed as an educational tool to improve understanding of Data Structures and Algorithms concepts through visual learning and interaction.
